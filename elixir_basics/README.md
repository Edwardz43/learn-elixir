# elixir_basics

Elixir 基礎學習專案 — 從零開始認識 Elixir。

適合有 JS/TS、PHP、Go、Java 經驗的開發者。

## 快速開始

```bash
# 編譯
mix compile

# 在 IEx 中互動
iex -S mix

# 執行測試
mix test
```

## 測試

```bash
mix test
```

所有範例都附有 doctest，可以直接在 IEx 中嘗試。
